Switch the DJ Claude audio backend to browser mode, which opens a browser tab for higher-quality audio playback.

## Usage

Call the `switch_audio` MCP tool from the `dj-claude` server with `mode: "browser"`.

## Examples

- `/dj-claude:browser`
- "switch to browser audio"
- "use browser for audio"

## Instructions

1. Call the `switch_audio` tool with `mode` set to `"browser"`.
2. Let the user know the audio backend has been switched to browser mode and a browser tab will open for audio playback.
